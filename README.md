## 로딩 플로우

```
┌───────────────┐
│  대화 시작    │
└───────┬───────┘
        │
        ▼
┌──────────────────────────────────────────────┐
│     CLAUDE.md 자동 로드                      │  ◄── 항상 (always_applied_workspace_rules)
│    (핵심 정보 + 명령어)                      │
└───────┬──────────────────────────────────────┘
        │
        ▼
┌──────────────────────────────────────────────┐
│    사용자 요청 분석                          │
└───────┬──────────────────────────────────────┘
        │
        ├──── "API 기능 구현해줘" ──────────────► read_file(tech_stack.md)
        │
        ├──── "프로젝트 목표가 뭐야?" ─────────► read_file(prd.md)
        │
        └──── "flutter run 해줘" ──────────────► 바로 실행 (CLAUDE.md에 명령어 있음)
```
